The Power of Bootstrap!

Originally created by and for Twitter, Bootstrap became open source in 2011. Bootstrap allows designers and developers to create websites quickly. Bootstrap is responsive by design and it also offers consistent design. It's easy to use and quick to learn. But most importantly it is a powerful toolkit that allows the developers to focus on well... developing, without having to worry about design.

So how does it work?

Well, it's really quite simple. You can download Bootstrap to get the complied CSS, JavaScript, and source code. Or as we've been learning you can simply copy and paste their starter template from their website into your .html file. Once the starter template is in your .html file, it's linked to Bootstraps' .css stylesheet giving us all their default settings.

Grid

Bootstrap uses the mobile first approach to their layouts using flexbox grid. It uses a twelve column system and is fully responsive. The grid supports 6 breakpoints, allowing the developer to control the container and column sizing behaviour by each breakpoint. Because they use the mobile first approach the breakpoints are based on min-width media queries which means they affect the breakpoint and all others above it.

Typography

Bootstrap wants the developer to be able to focus on producing quick, clean and styled responsive websites. It does this by giving us a default of stylized content. We don't have to worry about going into the CSS and changing all of our fonts. Of course we can still add custom fonts and styles to our hearts content, but with bootstrap we can put our main focus on making sure our websites work the way we intend first. The default font is "Helvetica Neue", Helvetica, Arial, sans-serif. We can also take advantage of the class system and easily add heading tags to paragraph tags and other heading tags.
